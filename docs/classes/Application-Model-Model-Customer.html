Model_Customer.php : 18

The Model object for the Customer page.

This class handles the retrieval and submission of data relating to Customers in the database. Its other MVC components are:

  • View: Customer.php
  • Controller: Controller_Customer

When Model_Customer::fetchModelData() is called (page load), it retrieves the table structure of the address table to allow the controller to build a form for it. When Model_Customer::sendModelData() is called, it either creates a new address record or updates an existing one. See Model_Customer::sendModelData() for specifics.
